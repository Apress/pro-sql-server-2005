This sample application shows you how to use the new Reporting Services ReportViewer controls in Visual Studio 2005.

Please note that you must have the AdventureWorks sample database, the AdventureWorks sample reports and install the GetProducts stored procedure from the Integration Services Product 1 sample in this chapter, installed to use this application.

To use the application, please follow these steps:

1) Double-click the Report Control Sample.sln file to launch Visual Studio 2005.

2) Once loaded, press ctrl-F5 to run the application without debugging.

3) When the application runs, it should load a server report in the top ReportViewer control, load a client-side report in the middle ReportViewer control and when you click the Programmatic button, programmatically load a client-side report in the bottom ReportViewer control.
